Project title: HarmoniOcean-AI
Problem statement:Create a technology driven solution to tackle pollution, biodiversity loss, and resources overuse in marine ecosystems.
proposed solution:This project uses AI/ML techniques to build a real-time detection system. It processes input data, runs predictions
using a trained model, and provides instant results through a user-friendly interface.
Tech Stack Used:Frontend: HTML, CSS, JavaScript
Backend: Python (Flask / FastAPI)
Machine Learning: TensorFlow / PyTorch / Scikit-learn
Database (if used): SQLite / MongoDB
Tools: Git, GitHub, VS Code
Installation/Run Instructions: 
1.git clone https://github.com/varu91490-ui/harmoniocean-ai.git
2.cd harmonyai
3.pip install -r requirements.txt
4.python app.py
